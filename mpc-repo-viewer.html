This repository contains free resources for customizing Akai MPC Software 2.11 and above, including:

  • Custom Plugin Skins - Redesigned UI themes for AIR and AKAI audio plugins
  • API Documentation - Technical references for plugin parameters and structures

  • Tutorials - Installation and creation guides

  1. Download the desired theme package
  2. Open the Air Plugins Themes folder on your computer
